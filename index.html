Block

src/index.js

Represents an immutable block of data that is uniquely referenced with a cid.

new Block(data: Uint8Array, cid: CID)
Parameters
data (Uint8Array) The data to be stored in the block as a Uint8Array.
cid (CID) The cid of the data
Example
const block = new Block(Uint8Array.from([0, 1, 2, 3]), new CID('...'))
